About Us

Somos un equipo de desarrolladores Fullstack, por el momento asumimos la division technologica de Almacenes Only.

Para la creacion de estos productos tecnologicos, dirigidos hacia Almacenes Only, utilizamos el stack de las siguientes tecnologias conocido mejor como MEAN:

  • Mongo DB: Base de Datos no relacionales
  • Angular: Javascript Framework for Front end
  • Express: Javascript Framework for Back end
  • Node: Javascript Development environment
